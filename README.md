# Project_PONG

## Requirements

### Software Requirements
 - PlatformIO

### Hardware Requirements
 - Arduino Uno R3
 - 2x Stepper Motor (JK42HS40-0404)
 - 2x L298 Based Motor Driver
 - 2x 12V 7,2W Solenoid

## Build & Deploy
```bash
$ pio run -e release -t upload
```
